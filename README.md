# 🤖 ConvoPilot

> Automate • Connect • Save Time

ConvoPilot is a premium WhatsApp automation platform built using Python and Flask. It allows users to schedule WhatsApp messages, track message history, and automate communication through WhatsApp Web.

---

## ✨ Features

- 📅 Schedule WhatsApp messages
- 📱 Indian phone number validation
- 📊 Message history with status tracking
- 🔄 Background scheduler
- 🎨 Premium dark UI
- 🗄️ SQLite database integration
- ⚡ Real-time status updates

---

## 🛠️ Tech Stack

- Python
- Flask

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ConvoPilot.git
cd ConvoPilot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```text
http://localhost:5000
```
